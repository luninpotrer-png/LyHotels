## 📝 Description / Descripción

**EN:** **LyHotels** is a professional web platform for hospitality management in Huaura, Peru. It allows hotel owners to list their properties while providing guests with a secure way to review and rate their stays.

**ES:** **LyHotels** es una plataforma web profesional para la gestión hotelera en Huaura, Perú. Permite a los dueños de hoteles listar sus propiedades, mientras brinda a los huéspedes una forma segura de reseñar y calificar sus estancias.

## 🛠️ Tech Stack / Tecnologías

- **Backend:** Flask (Python 3.13) & SQLAlchemy.
- **Images:** Cloudinary API.
- **Frontend:** Jinja2, CSS3 (Modern Grid/Flexbox).
- **Database:** SQLite / PostgreSQL.

## 📦 Setup / Instalación

1. `git clone https://github.com/tu-usuario/LyHotels.git`
2. `python3 -m venv .venv && source .venv/bin/activate`
3. `pip install -r requirements.txt`
4. Create `.env` with Cloudinary keys.
5. `python app.py`
